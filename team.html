# 웹개발

사이트의 테마는 주제인 '보안'과 어울리게끔 배경에 #0a0e27 색상을 사용하였고, #03ff40 색상을 사용하여 핵심 단어를 강조했습니다. 전체 페이지를 미디어쿼리를 활용하여 반응형으로 구성함으로써 어떤 기기에서든 불편함 없이 접속할 수 있도록 하였습니다.

메인 페이지인 index.html에서는 웹사이트의 주제를 명확하게 나타내었고, 관련 그림을 넣어 쉽게 이해할 수 있도록 하였습니다.

사이트의 핵심인 security.html에서는 웹 보안, 앱 보안, 네트워크 보안으로 3개의 카테고리를 구성하여 각 주제마다 3개씩 총 9개의 주제를 설명했습니다.

팀원 소개 페이지인 team.html에서는 팀원의 이름과 전공 등을 소개하고, 각 팀원의 Github, 블로그, 포트폴리오에 접근할 수 있도록 하였습니다.
