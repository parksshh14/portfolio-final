# 웹브라우저

개인 프로젝트를 위해 HTML, CSS, Javascript를 이용한 웹사이트를 제작하였습니다.

CSS를 사용하여 메인 내용이 들어갈 부분을 웹 브라우저 모양으로 구성하였습니다. 또한, 내용이 많아도 화면 전체가 스크롤 되는 것이 아닌, 웹 브라우저 모양 안에서만 스크롤되도록 하였습니다.

구현한 웹 브라우저의 하단에는 Javascript를 이용하여 실시간으로 시계가 작동하도록 하였습니다.

외부 프레임워크를 이용하여 간단한 댓글 기능을 구현하였습니다.
